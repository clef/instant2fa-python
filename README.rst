===============================
instant2fa
===============================

Instant2FA Python bindings.

Installation
------------

Install it via PyPI:

.. code-block:: shell 

    pip install instant2fa

Usage
-----

.. code-block:: python

    import instant2fa

Development
-----------

To start development, run:

.. code-block:: shell

    $ git clone git@github.com:clef/instant2fa-python.git instant2fa
    $ cd instant2fa
    $ make develop
